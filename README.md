## Description

Service app for rent car.

## Installation

```bash
$ npm install
```

## Running the app
Create a `.env` file then copy values from `.env.example`
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
